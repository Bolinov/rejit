Regexp: (abcd---#|abcd___#)
Search for an alternation of two words sharing a common prefix.
The character '#' is not present in the text searched to ensure there is no match. We cannot use a trailing '$' because rejit would use it as the fast forward element.
Following benchmarks containing '#' use the same trick.
Re2 spots the common prefix and uses memchr() to speed up the search. Rejit does not do that yet.
